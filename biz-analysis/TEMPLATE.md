# Biz Analysis Skill — 快速应用模板

直接复制这个模板，填充你的业务信息，然后调用 Skill。

---

## 模板

```
@.claude/skills/biz-analysis/SKILL.md 帮我深度分析 [业务名称] 的 [核心问题]。

【背景】
- 产品是什么：[一句话描述，如"天天练是K12英语学习产品"]
- 核心变现模式：[如何赚钱，如"活动付费、会员订阅"]
- 目标用户：[谁在用，如"小学生、家长"]
- 当前核心KPI：[具体数字，如"付费率4%、人均时长180s"]
- 当前卡点：[遇到什么问题，如"新图进入热图转化率仅15%"]

【目标】（按优先级排列）
目标1（最核心）：[主要问题，如"提升新图发现效率"]
目标2：[次要问题，如"优化热图更新频率"]
目标3：[附加问题，如"建立创作者激励SOP"]

【我会提供以下材料】
[列出你有的文档，如"活动数据看板截图、用户反馈文档、AB实验记录"]

【约束/已知信息】
- 产品类型：[教育/游戏/电商/其他]
- 投放平台：[Meta/Google/抖音/TikTok/其他]
- 预算规模：[如"月投放50万"]
- 团队限制：[如"新代理商经验不足"或"历史数据不完整"]
- 已有假设：[如"假设300s门槛过高"]
- 团队的疑问：[如"个性化推荐为什么迟迟不上线？"]

【要求】
- 每个结论必须有数据支撑（内部数据或竞品数据）
- 好的案例和差的案例各举3个，并分析根因
- 提炼可复用的规律（铁律）
- 给出竞品对标（至少3个竞品的实际数据）
- 优先级分层：本周 / 2-4周 / 1-3个月
```

## 📝 填写示例

### 示例1：天天练地图推荐系统

```
@.claude/skills/biz-analysis/SKILL.md 帮我分析天天练UGC地图推荐系统

【背景】
- 产品：英语天天练UGC地图生态
- 核心变现：用户时长 → 转化课程/会员
- 目标用户：K12学生
- 当前核心KPI：人均停留时长（热图>600s，次热图~300s，新图<150s）
- 当前卡点：新图进入热图池门槛300s，转化率仅15%

【目标】
1. 核心问题：提升地图推荐的人均停留时长（目标+20%）
2. 次要问题：提高优质新图的发现效率（目标转化率30%）
3. 附加问题：明确个性化推荐上线的卡点

【材料】
- 系统设计文档（已提供）
- 0409-0415大盘数据（待补充）

【约束】
- 产品类型：教育
- 算法能力：有dcn_v2模型，每天在线更新
- 团队限制：个性化方案v2已准备，但未上线
- 已有假设：300s门槛可能过高，次热图固定曝光浪费流量

【要求】
- 每个结论有数据支撑（请补充0409-0415大盘数据）
- 好的差的案例各3个，分析根因
- 提炼可复用规律（铁律）
- 竞品对标（Roblox/王者/迷你世界）
- 优先级：本周/2-4周/1-3个月
```

## 💡 使用技巧

1. **信息越多，结论越精准**：背景里的数字越具体，AI 的分析越贴合实际
2. **目标要分优先级**：AI 会按权重分配分析深度
3. **把疑问写进去**：团队的争议点、假设，AI 会重点处理
4. **材料列清楚**：告诉 AI 你有哪些文档，它会在分析中引用
